Gemini and API-Football calls now run through this site's own /api backend, so no key ever touches your browser. If a request fails with a "server is missing" error, the site owner needs to add GEMINI_API_KEY and API_SPORTS_KEY in Vercel → Project Settings → Environment Variables, then redeploy. Three more are optional: GROQ_API_KEY (free, no card — automatic backup chat AI for when Gemini's daily quota runs out), ELEVENLABS_API_KEY + ELEVENLABS_VOICE_ID (primary voiceover — studio quality, with correct pronunciation), and GOOGLE_TTS_API_KEY (backup voiceover if ElevenLabs isn't set up or its quota runs out — needs a Google Cloud billing account with a card on file, though usage stays free under 1M characters/month). Without any of the three, voiceover still works via the browser's own built-in voice.
Your workspace — search results, chat history, script draft, and voice settings — autosaves locally in this browser, so reopening the page picks up right where you left off (screenshots/files you attached in chat aren't kept, since images are too large for local storage; everything else is).

Chat With Your Match Analyst
Talk this through instead of a single search. It automatically sees whatever's in the Verified Match Facts panel above, so it won't re-ask for anything already found — tell it what you already have, and it'll ask for whatever stats are still missing — possession, pass completion, chances created, big chances, shots on/off target, interceptions, touches, for either team or a specific key player. Attach as many screenshots as you need at once (from Sofascore, WhoScored, etc.) and it'll read the numbers straight off each image and read them back to confirm. You can also upload a stats export directly — an .xlsx or .csv file (e.g. from Gemini/Sheets) — instead of screenshotting it.
